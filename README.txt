Чтобы использовать программу нужно иметь Python 3 версии и выше.Так же скаччать библиотеку tkinter.
1. Cохранение файла - нужно выбрать файл и вы получите его id.
2. Удаление файла - нужно написать id и файл будет удален.
3. Переименовать файл - нужно написать новый id и старый через пробел.
4. Список файлов - по нажатию выдает id файлов и пути до них.
5. Получение пути файла - нужно ввести id файла и вы получите путь до него.
