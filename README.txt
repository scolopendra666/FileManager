Чтобы использовать программу нужно иметь Python 3 версии и выше.Так же скаччать библиотеку tkinter.
1. Cохранение файла - выберите файл и вы получите его id.
2. Удаление файла - введите id и файл будет удален.
3. Переименовать файл - введите новый id и старый через пробел.
4. Список файлов - по нажатию на кнопку выдает id файлов и пути до них.
5. Получение пути файла - введите id файла и вы получите путь до него.
6. Получение списка пути файлов по id - введите id файлов через пробел.
7. Резервное копирование - введите id файла который нужно сохранить.
